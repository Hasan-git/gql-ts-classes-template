# gql-ts-classes-template


 >> cd boilerplate
 >> yarn
 >> npm i -g graphql
 >> npm i @graphql/types
 >> yarn build


Usage
 >> copy dist to your project
 >>node_modules/.bin/gql-gen --schema [path/to/graphql/schema.graphql] --template [Path/to/generated/template]  --out [./path/output.ts] [./src/app/blocks/graphql/**/*.gql]
